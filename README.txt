Daily Greeting - Random Message Generator
Version - 1.0.0
node.js / Command line program

This program is used to generate a positive message to the user.
The message will include the following:
- A basic greeting
- A joke
- A compliment

System Requirements
-------------------
node.js v14.15.4. or newer

To run program
--------------
- Make sure node.js is installed and accessable from your command line.
- Make sure you know the location of the program folder.
- From the command line terminal, open the program folder.
    Ex: user@PC-NAME MINGW64 /C/Users/username/programfiles/daily_greeting
- Use node.js to run greet_me.js
    Ex: $ node greet_me
